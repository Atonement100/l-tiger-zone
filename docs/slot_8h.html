Compare the positions (x,y coordinates) of two, adjacent slots.

Parameters
aFirst slot.
bSecond slot.
Returns
negative value if a is to the left of or above b, 0 if a and b are the same slot, or a positive value if a is to the right of or above b.

Returns the relative position of a to b; x position takes precedence over y position. If a comes before b, -1 is returned. If a is the same position as b, 0 is returned. If a comes after b, 1 is returned.

Get all slots adjacent to the input slot.

Parameters
adjslot pointer to hold the adjacent slots.
sThe slot for which the the adjacent slots are found.
Returns
An array of the 4 slots adjacent to the input slot. 0 for above, 1 for right, 2 for below, 3 for left.

Fetches the slots adjacent to the given slot and stores them into adj in the order of top, right, bottom, left.
